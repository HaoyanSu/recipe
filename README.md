# 🍳 食谱探索 (Recipe Finder)

一个简洁易用的食谱搜索小工具，支持自然语言输入，帮你快速找到想吃的美食。

## ✨ 功能特点

### 1. 自然语言搜索
- 用自然语言描述你想吃的东西
- 支持描述口味、做法、食材
- 支持指定禁忌（如"不要放葱"）

### 2. 网红食谱参考
- 自动搜索相关的网红食谱
- 展示来自小红书、抖音、下厨房等平台的热门食谱

### 3. 智能排序
- **推荐排序**：根据匹配度推荐
- **最快手**：按烹饪时间从短到长排序
- **最低卡**：按热量从低到高排序

### 4. 直观的食谱展示
- 食材清单一目了然
- 营养成分可视化
- 详细的步骤说明
- 每步配有图片和小贴士
- 横滑浏览每个步骤

### 5. 对话式调整
- 继续对话来调整食谱
- 快速添加限制条件
- 实时更新推荐结果

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 🛠️ 技术栈

- **React 18** - UI 框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **TailwindCSS** - 样式框架
- **Lucide React** - 图标库
- **Framer Motion** - 动画库

## 📁 项目结构

```
src/
├── components/          # React 组件
│   ├── SearchInput.tsx     # 搜索输入框
│   ├── RecipeCard.tsx      # 食谱卡片
│   ├── RecipeDetail.tsx    # 食谱详情弹窗
│   ├── SortTabs.tsx        # 排序标签
│   ├── ChatRefine.tsx      # 对话调整
│   └── WebSearchResults.tsx # 网络搜索结果
├── data/
│   └── mockRecipes.ts   # 模拟数据
├── types.ts             # TypeScript 类型定义
├── App.tsx              # 主应用组件
├── main.tsx             # 入口文件
└── index.css            # 全局样式
```

## 🎨 设计特点

- 温暖的配色方案（香料色、奶油色、木质色）
